1、贝叶斯公式推导

【已知】假设学校总人数U个

穿长裤的（男生）：U*P(boy)*P(pants|boy)

P(boy)是男生的概率 = 60%

P(pants|boy)是条件概率，即在boy条件下穿长裤的概率，所有男生都穿长裤 = 100%

穿长裤的（女生）：U*P(girl)*P(pants|girl).

【要求】穿长裤的人里面有多少女生: P(girl|pants)

穿长裤的总数：U*P(boy)*P(pants|boy) + U*P(girl)*P(pants|girl).

所以，

P(girl|pants) = U*P(girl)*P(pants|girl) / [U*P(boy)*P(pants|boy) + U*P(girl)*P(pants|girl)]

化简，约掉U

P(girl|pants) = P(girl)*P(pants|girl) / [P(boy)*P(pants|boy) + P(girl)*P(pants|girl)]

分母为P(pants)

【结果】P(girl|pants) = P(girl)*P(pants|girl) / P(pants)

案例1——拼写纠错

【问题】我们看到用户输入了一个不在字典中的单词，我们需要去猜测“用户想输入的是什么”

即求 P(我们猜测他想输入的单词 | 他实际输入的单词)

用户实际输入的单词为 D

猜测1：P(h1|D) , 猜测2：P(h2|D), 猜测3：P(h3|D) …

统一为：P(h|D)

【结果】P(h|D) = P(h) * P(D|h) / P(D).

对不同的具体猜测h1、h2、h3 ，P(D)都是一样的，所以在比较 P(h1|D) 和 P(h2|D) 可以忽略 P(D)

P(h)为先验概率：这里为h1、h2、h3的词频统计

P(D|h) 为自定规则：这里为用户输入的信息熵差别

案例2——垃圾邮件过滤

【问题】.

给定一封邮件，判定它是否属于垃圾邮件

【1.数学表达】.

D表示这封邮件，D由N个单词组成，用h+表示垃圾邮件，h-表示正常邮件

P(h+|D) = P(h+) * P(D|h+) / P(D)

P(h-|D) = P(h-) * P(D|h-) / P(D)

先验概率P(h+)和P(h-)易得：即邮件库中垃圾邮件和正常邮件的比例

【2.扩展D的表达】.

D里有N个词，即是垃圾邮件情况下，D恰好按d1、d2、d3排序的概率

P(D|h+) = P(d1,d2,... dn|h+) = P(d1|h+) * P(d2|d1,h+) * P(d3|d2,d1,h+) * ...

【3.化简：朴素贝叶斯】

假设di与di-1是完全条件无关的（朴素贝叶斯假设特征之间相互独立，互不影响

化简为：P(d1|h+) * P(d2|h+) * P(d3|h+) * ...

【4.结果】只要统计di这个单词在垃圾邮件中出现的词频即可

计算 d1、d2、d3在垃圾邮件中的词频；

计算 d1、d2、d3在正常邮件中的词频；

二、BN贝叶斯网络

1、BN的数学表达 + 每个节点的概率=P (X | parents(X) )

组成条件：

  • 一组变量 X={ X1, X2, … Xn }
  • 一个编码了条件独立信息的有向无环图S，每个节点代表一个变量X_i
  • 一系列局部概率信息P

贝叶斯网络的独立性假设：每个节点的概率只依赖父节点

BN的数学表达，联合概率可写成局部概率的乘积：

p(x) = PRODUCT( p(xi|pai) )

如下图：P(R,W,S,C) = P(R)*P(C)*P(W|C,R)*P(S|W)

2、建立BN的步骤

（1）明确想解决的问题，依据条件建立网络结构

（2）采集足够的观测数据

（3）根据数据更新节点参数（各节点条件概率）

贝叶斯是可解释网络（对比ML、DL），适用概率推断问题
