Author: Dr.yuan Date: June 26, 2025  12:40:00

一、如何写CSS

1、内联样式

<p style="color:red;font-size:12px">这里文字是红色。</p>

css样式代码要写在style=””双引号中，如果有多条css样式代码设置可以写在一起，中间用分号隔开。

2、嵌入style标签式

嵌入式css样式必须写在<style></style>之间，并且一般情况下嵌入式css样式写在<head></head>之间

3、引入外部CSS式

<link href="base.css" rel="stylesheet" type="text/css" />

css样式文件以“.css”为扩展名，link写在内，使用标签将css样式文件链接到HTML文件内

  • css样式文件名称以有意义的英文字母命名，如 main.css。
  • rel=”stylesheet” type=”text/css” 是固定写法不可修改。
  • 标签位置一般写在标签之内。

三种写法的优先级：就近原则，内联式 > 嵌入式 > 外部式

**写入style标签与外部CSS的内容相同：选择器class 和 id **

4、盒模型

三层结构：

  • margin 外部空间
  • border
  • padding 盒内空间

5、color

（略）

6、opacity & border radius

opacity: 0.5; opacity取值0~1的透明度

border-radius: 10px; 矩形的圆角

7、选择器

8、布局

（1）float和display

float：将元素图层浮上来

display：用来改变元素块的属性

（2）position

  1. position: fixed; 固定定位，于屏幕固定位置（独立图层）
  2. position: relative; 相对定位，相对于页面中元素原来的位置（原来的面积还存在）
  3. position: absolute; 绝对定位，相对于页面或上一层被定位的元素（独立图层）

（3）flex

display: flex; 写在style内， 从左至右，同一行-按比例显示方式

  • flex: 1; flex: 2; flex: 3; 按数字比例划分范围
  • flex-wrap: wrap; 组内元素按浏览器宽度自动换行
  • flex-direction: column; 按纵轴方向显示
  • justify-content: center; 按主轴居中

9、animation

从时间点1的CSS，转换到时间点2的CSS， 写在style内

  1. 状态的CSS定义，颜色、位置等
  2. animation的属性
