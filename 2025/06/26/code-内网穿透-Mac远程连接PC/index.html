3、-获取公网IP

获取公网IP的方法：拨打运营商的客服电话，然后找人工服务。跟客服说我家里的宽带需要一个公网ip地址。理由：在家里搭建网站，需要用公网IP地址或者说家里安装远程监控，需要用公网IP地址。这两条理由都可以，一般2~3个小时就会给你解决IP地址问题，然后有关人员会给你打电话告诉你你的IP地址已经是公网IP了。
