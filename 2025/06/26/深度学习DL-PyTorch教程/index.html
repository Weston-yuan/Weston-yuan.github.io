四、pytorch实现CNN网络

PyTorch 是一个用于在 Tensor 对象上执行动态计算图的系统，其行为类似于 numpy ndarray。 它配备了强大的自动微分引擎，无需手动反向传播.

准备工作：挂载硬盘，切换GPU，下载、分割数据集，预处理

1、实现方式1——准系统 PyTorch：，我们将直接使用最低级别的 PyTorch 张量

调用pytorch的API实现relu层，two_layer_fc，three_layer_convnet，Initialization， Training a ConvNet

调用pytorch的API实现relu层，此实现使用 PyTorch 张量上的操作计算前向传递，并使用 PyTorch autograd 计算梯度。

tensor张量 = numpy数组. 图像数据通常存储在形状为 N x C x H x W 的张量中，其中：

N 是数据点的数量
C 是通道数
H 是以像素为单位的中间特征图的高度
W 是以像素为单位的中间特征图的高度

2、实现方式2——PyTorch 模块 API，我们将使用 nn.Module 来定义任意神经网络架构

调用pytorch的nn.Module API实现：TwoLayerFC、ThreeLayerConvNet、训练过程

PyTorch 为您提供 nn.Module API 来定义任意网络架构，同时为您跟踪每个可学习的参数。PyTorch 还提供了 torch.optim 包，它实现了所有常见的优化器，例如 RMSProp、Adagrad 和 Adam。

1.子类 nn.Module。 为您的网络类起一个直观的名称，例如 TwoLayerFC。

2.在构造函数 init() 中，将您需要的所有层定义为类属性。

3.在 forward() 方法中，定义网络的连接性。

定义 Module 子类后，您可以将其实例化为一个对象并像第二部分中的 NN 前向函数一样调用它。

3、实现方式3——PyTorch Sequential API，我们将使用 nn.Sequential 非常方便地定义一个线性前馈网络

nn.Sequential 的容器模块打包3个步骤：子类nn.Module，在init中将层分配给类属性，并在forward()中逐层调用
