Author: Dr.yuan Date: June 30, 2025  21:00:20

docker快捷键

基础知识

1、Dockerfile是一个自动化脚本，用于创建镜像（类似在虚拟机中安装软件的过程） 2、Image镜像：类似虚拟机的快照，里面包含待部署的应用程序+所有依赖项-库软件 3、通过镜像创建不同的容器：容器相当于一台台的虚拟机，每个容器独立运行-相互不影响

示例practice

1、创建dockerfile
在官方市场寻找dockerfile模板，
cd到目标目录下：

2、【创建镜像】终端：docker build -t XXX(镜像名):tag . 3、【运行镜像——创建容器】终端：docker run -p 80: 5000 -d XXX(镜像名) 将容器端口5000映射到本地浏览器端口80，-d 让容器在后台运行

4、定义yml文件，使多个容器协作

运行所有容器：docker compose up
停止所有容器：docker compose down
